# Sentiment Analysis API

A REST API for sentiment analysis using Logistic Regression and TF-IDF vectorization.

## Features
- Predicts sentiment (positive/negative) from text
- Returns confidence scores and probabilities
- Simple RESTful interface
- Ready for deployment

## Live Demo
API Endpoint: `https://sentiment-api.onrender.com/predict`

## Usage

### Local Development
```bash
pip install -r requirements.txt
python app.py
